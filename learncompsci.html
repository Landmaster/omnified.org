Tested Teaching

Learncompsci.org is an online platform in which computer science learning materials are provided for free. The intention of this service is so that anybody can come and access slideshows and video tutorials about various topics. Workshops led by F[Space] will also to be posted onto this website, in both slideshow and mp4 form. All presentations are reviewed by a board of computer scientists. This project is still under construction, we aim for this site to contain 9+ courses by the end of May 2019. This includes machine learning, data science, C IOT, Scratch, CAD, career entry point, python, web development, and more.
